ABOUT ME

프론트엔드 웹개발자를 꿈꾸는 김수진입니다.

컴퓨터공학과를 졸업한 후 방송정보국제교육원에서 스마트웹 국비 과정을 들으며 퍼블리싱과 프론트엔드 개발의 기본을 다졌습니다. HTML, CSS, SASS, Javascript, JQuery를 사용하여 웹페이지를 구축할 수 있으며, Git에 대한 이해도가 높아 협업도 가능합니다. 현재 React와 자바스크립트를 매일같이 공부하며 프론트엔드 개발자로서의 자질을 갖추기 위해 노력하고 있습니다. 다른 무엇보다 기초가 탄탄히 다져져있는 개발자가 더 좋은 코드를 만들 수 있다고 생각합니다. 기술의 변화를 유연하게 받아들이고 그에 발맞춰 함께 발전하는 개발자가 되는 것이 목표입니다.

프로젝트 소개 : Be-us

비어스는 가장 처음 진행한 프로젝트로, 기존의 로이비 브랜드를 새로운 브랜드로 만들어내어 리뉴얼을 진행한 웹페이지입니다. 기존의 디자인에서 브랜드 소개 영역, footer영역, 사용되는 font 등을 변경하였습니다. 또한, flex를 적극 사용하여 mobile, tablet, pc에 각각 적용되는 반응형 웹페이지를 구현하였습니다.

구분 : 개인 프로젝트

사용 언어 : HTML5, CSS3, SASS, Javascript

제작 페이지 : 메인 페이지, 로그인페이지
